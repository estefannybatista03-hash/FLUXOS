# FluxOS - Referência de Design

## Análise Visual do Site Original

### Paleta de Cores
- **Sidebar**: Roxo vibrante (gradiente roxo/azul) - cor primária
- **Fundo**: Branco/cinza claro
- **Destaques**: 
  - Verde (Concluídas)
  - Roxo/Magenta (Performance da Equipe)
  - Azul/Roxo (Gráficos)
  - Rosa (Atrasadas)

### Layout
- **Sidebar esquerda**: Navegação vertical com ícones e labels
- **Conteúdo principal**: Grid responsivo com cards
- **Header**: Branco com botões de usuário e tema

### Componentes Principais
1. **Cards de Métrica**: 4 cards com ícones coloridos, números grandes e variação percentual
2. **Gráfico de Pizza**: "Ordens por Status" com cores correspondentes
3. **Card de Performance**: Fundo roxo/magenta com circular progress (87%)
4. **Gráfico de Barras**: "Conclusões vs Atrasos" com dados mensais
5. **Gráfico de Linha**: "Receita do Mês" com valores monetários
6. **Listas**: Agenda do dia, Últimas Ordens, Notificações
7. **Atalhos Rápidos**: Botões de ação rápida na base

### Tipografia
- Títulos: Fonte sem serifa, peso bold
- Corpo: Fonte sem serifa regular
- Números grandes: Muito bold

### Ícones
- Ícones coloridos em cada métrica
- Ícones na sidebar para navegação
- Ícones em listas (status badges)

### Interatividade
- Botão "Alternar para modo escuro" (tema)
- Botões com ações (Ver relatório, Ver agenda, etc.)
- Menu de usuário (JS - João Silva)
